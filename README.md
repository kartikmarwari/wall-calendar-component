# wall-calendar-component

# 📅 Interactive Wall Calendar

## 🚀 Overview

A responsive and interactive wall calendar component built using React.
Inspired by physical wall calendars with a modern UI.

## ✨ Features

* Date range selection (start → end)
* Notes section (local storage)
* Dark / Light theme
* Dynamic monthly images
* Fully responsive design

## 🛠 Tech Stack

* React (Vite)
* CSS (custom styling)
* LocalStorage

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 📱 Responsiveness

* Desktop → side-by-side layout
* Mobile → stacked layout

## 💡 Design Decisions

* Used CSS Grid for layout
* Used localStorage (frontend-only constraint)
* Focus on UX + smooth interactions

## 🔮 Future Improvements

* Per-day notes
* Drag date selection
* Animations
